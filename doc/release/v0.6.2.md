# v0.6.2 — 추가 매크로 Toggle 변환

**날짜**: 2026-02-23

## 변경 사항

### tabs-group / tab-pane 매크로 변환

- Confluence `tabs-group` + `tab-pane` 매크로를 Notion toggle(`<details>/<summary>`) 블록으로 변환
- 기존 `ui-tabs` 오버라이드와 별도로, 실제 콘텐츠(이미지·테이블 등)를 toggle 안에 포함
- `tab-pane`이 개별 호출될 경우 noop 처리하여 중복 방지

### horizontal-nav 제거

- `horizontal-nav-group` / `horizontal-nav-item` 매크로 출력을 제거
- toggle 변환 시 탭 헤더 네비게이션이 불필요하므로 빈 문자열 반환

### ui-expand Toggle 변환

- `ui-expand` (펼치기/접기) 매크로를 toggle 블록으로 변환
- `data-title` 속성에서 제목을 추출하여 `<summary>`에 표시

## 변경 파일

| 파일 | 내용 |
|---|---|
| `pipeline/converter_overrides.py` | `_convert_tabs_group`, `_noop`, `_convert_horizontal_nav`, `_convert_ui_expand` 함수 추가 및 `_DIV_MACRO_OVERRIDES` 등록 |
