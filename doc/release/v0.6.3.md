# v0.6.3 — 마크다운 전처리 품질 개선 (5건)

**날짜**: 2026-02-24

## 변경 사항

### 1. 인라인 헤딩 분리

- `text## heading` 처럼 이전 텍스트에 줄바꿈 없이 붙어 있는 헤딩을 자동으로 분리
- `</details>## heading`, `callout텍스트## heading` 등 모든 케이스 대응
- `#` 내부 false positive 방지를 위해 lookbehind에서 `#` 제외 (`(?<=[^\s#])`)

### 2. 3중 URL 중복 제거

- Confluence 변환 시 생성되는 `<url>url (url)` 패턴을 `[url](url)` 단일 링크로 정리

### 3. {{BR}} 매크로 처리

- Confluence `{{BR}}` 매크로를 줄바꿈(`\n`)으로 변환
- `convert_alerts_to_callouts` 함수에서 `{{BR}}` 대신 `\n`을 사용하여 callout 내 줄바꿈 정상화

### 4. Callout 내부 `<details>` 토글 분리

- `> [!NOTE]` 블록 내부에 `<details>/<summary>` 가 포함된 경우, 토글을 callout 밖으로 추출
- callout에는 텍스트만 남기고, 토글 블록은 별도로 렌더링

### 5. H4-H6 → H3 변환

- Notion은 H1-H3만 지원하므로 `####`, `#####`, `######` 헤딩을 `###` (H3)으로 통일

### 6. Toggle 내부 콘텐츠 정상 파싱

- `<details>/<summary>` 를 `TOGGLE_START::/TOGGLE_END` 텍스트 마커로 변환
- `notion-markdown` 이 내부 마크다운(헤딩·인용·이미지)을 정상 파싱
- 후처리에서 마커를 toggle 블록으로 조립 (중첩 지원)

### 7. 첨부파일 섹션 제거

- Confluence 첨부파일 목록 테이블(중복 데이터)을 자동 제거

### 8. 잔여 HTML 태그 제거

- `</details>` 등 orphaned HTML 태그 paragraph 블록 자동 제거
