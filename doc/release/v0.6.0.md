# v0.6.0 — 소스 가독성 정리

날짜: 2026-02-23

기능 변경 없이 코드 가독성과 유지보수성을 개선하는 리팩토링.

---

## 데드 코드 제거

- `notion_postprocessor.py`: `parse_bold_rich_text()` 함수 제거 (v0.5.0에서 `parse_callout_content()`로 대체 후 미사용)
- `notion_postprocessor.py`: `_BOLD_SPLIT_RE` 정규식 제거 (위 함수 전용이었음)
- `notion_postprocessor.py`: `extract_local_images` 호환 별칭 제거 (외부 참조 없음)
- `block_utils.py`: `filter_invalid_images` 호환 별칭 제거 (외부 참조 없음)
- `md_preprocessor.py`: 미사용 `VIDEO_EXTENSIONS` 상수 제거 (정규식이 확장자를 직접 포함)

## 네이밍 통일

- JSON 키·변수명 `local_images` → `local_media`로 통일 (이미지+비디오 포함을 정확히 표현)
  - `markdown_to_notion.py`: 변수명, JSON 키, 출력 메시지 갱신
  - `upload.py`: 기존 output 하위 호환을 위해 `local_media` 키 우선, `local_images` 키 폴백 지원
  - `getting-started.md`: 예시 JSON 갱신

## 상수 중복 해소

- `_VIDEO_EXTENSIONS`를 `md_preprocessor.py`에 단일 정의하고 `notion_postprocessor.py`에서 import
- 동일한 frozenset이 두 파일에 각각 정의되어 있던 문제 해소

## 미사용 변수 정리

- `notion_postprocessor.py`: `_alt` → `_`로 변경 (regex split 결과 중 미사용 alt text)

## 문서 갱신

- `getting-started.md` 프로젝트 구조 트리에 누락 파일 추가:
  - `pipeline/converter_overrides.py`, `pipeline/md_preprocessor.py`, `pipeline/notion_postprocessor.py`, `upload/block_utils.py`
- 실제 존재하지 않는 `__init__.py` 항목 제거
- 핵심 파일 설명 테이블에 위 파일들 역할 설명 추가
